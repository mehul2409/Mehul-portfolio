# Mehul Gosavi - Portfolio

A high-performance, dark-themed portfolio website designed for a Distributed Systems Engineer & Quant Developer. Built with modern web technologies to showcase technical depth, low-latency enginering, and algorithmic trading expertise.

## üöÄ Features

*   **"Dark Quant" Aesthetic**: Minimalist, deep black theme with neon green/blue accents (`#00ff9d`), inspired by terminal interfaces and trading platforms.
*   **Interactive Market Grid**: A custom HTML5 Canvas background that simulates a distributed node network.
    *   **Market Data**: Displays real-time simulated tickers (SPX, BTC, NVDA, etc.) with price fluctuations.
    *   **Candlestick Interactions**: Reacts to mouse movement with Red (Bearish) and Green (Bullish) candlestick graphics.
*   **Typewriter Hero Section**: Dynamic, cycling text highlighting key roles: "Architecting high-performance distributed systems".
*   **Visual Navigation**:
    *   **Scroll Spy**: Automatically highlights the active section in the navbar.
    *   **Bi-directional Navigation**: "Back to Top" (Bullish Candle) and "Go to Bottom" (Bearish Candle) floating buttons.
*   **Comprehensive Sections**:
    *   **Experience**: Vertical timeline of professional history (Cyble etc.).
    *   **Skills**: Categorized technical arsenal (Algorithms, Libraries, Infrastructure).
    *   **Projects**: Grid layout showcasing GitHub repos with "Private" and "Featured" indicators.

## ‚ö° Deployment & Setup

### Prerequisites
*   Node.js (v18+ recommended)
*   npm

### Installation

```bash
# Clone the repository
git clone https://github.com/mehul2409/mehul-portfolio.git

# Navigate to directory
cd mehul-portfolio

# Install dependencies
npm install
```

### Development

```bash
# Start local dev server
npm run dev
```

### Build for Production

```bash
# Create optimization production build
npm run build
```
